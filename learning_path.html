LLM 算法岗学习路线

从零基础到可投递 · 学习 + 项目双线并行

📅 20 周 · 40 小时/周 💻 4090 台式机 + 5070 笔记本 + MacBook 🎯 LLM 算法岗

🎯 目标

从仅有基础编程经验出发，通过 20 周系统学习 + 6 个实战项目，达到可投递 LLM 算法岗的水平。

📐 方法论

以项目驱动学习，每个阶段至少完成 1 个可展示的项目。不会的先问大模型，边做边学。

💡 核心理念

不需要"学完再做"。基础知识只要够用就立即上项目，在实战中补坑效率远高于纯理论啃书。

📦 最终产出

6 个 GitHub 项目 + 2 篇技术博客 + 1 份量化简历 + HuggingFace 模型主页作品集。

阶段 周数 核心主题 项目产出
一·地基 1-4 Python/PyTorch/数学/API 命令行AI助手
二·NLP & Transformer 5-8 Tokenizer/Transformer/HF/RAG 文档问答RAG
三·大模型核心 9-12 架构/微调/PEFT/数据集 Qwen3 微调
四·进阶对齐与部署 13-16 DPO/推理优化/Agent/全流程 完整微调流水线 + Agent
五·前沿与求职 17-20 MoE/GRPO/面试/简历 技术博客 + 作品集
